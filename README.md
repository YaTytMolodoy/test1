Тестовое задание
Стек технологий:
Vite
React для SPA,
React Router для маршрутизации,
SCSS для стилизации элементов,
Axios для удобности работой запросов,
MockAPI для тестовых пользователей и изображение бекенда
Effector в качестве стейтменеджера и уменьшить колво логики в UI
Ant Design для иконок

Были реализованы страницы
Главная, Личный кабинет, Контакты

Функционал:
Авторизация: была реализована через effector и имитации backend MockAPI
Модалка была реализована отдельным компонентом
Токен в локалстораж также истользуется
Валидация была реализована

Инструкция запуска:
Выберите папку для установки репозитория
cd путь/к/папке
Далее пропишите к терминале git clone https://github.com/YaTytMolodoy/test1.git
После этого зайти в папку репозитория cd .\test1
Прописать команду npm install
После установки всех пакетов, npm run dev
