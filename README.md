Тестовое задание
Стек технологий:
React для SPA,
React Router для маршрутизации,
SCSS для стилизации элементов,
Axios для удобности работой запросов,
MockAPI для тестовых пользователей и изображение бекенда
Effector в качестве стейтменеджера и уменьшить колво логики в UI
Ant Design для иконок

Главная, Личный кабинет, Контакты

Авторизация: была реализована через effector и имитации backend MockAPI
Модалка была реализована отдельным компонентом
